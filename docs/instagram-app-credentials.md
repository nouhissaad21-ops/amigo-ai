# Instagram Business Login credentials

AmiGo uses the dedicated Instagram App ID and Instagram App Secret shown under **Instagram > API setup with Instagram login** in Meta for Developers. These values are different from the Facebook/Meta App ID and secret.

Render environment variables:

- `INSTAGRAM_APP_ID`
- `INSTAGRAM_APP_SECRET`

OAuth redirect URI:

`https://amigo-ai-2d71.onrender.com/api/integrations/instagram/callback`
